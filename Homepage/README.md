# Suno Community Platform | 수노 커뮤니티 플랫폼

PHP + SQLite 기반 음악 커뮤니티 웹 애플리케이션입니다.
A music community web application built with PHP + SQLite.

## 설치 | Setup

1. PHP 8+ 필요
2. `homepage.zip` 압축 해제
3. `php -S localhost:8000` 실행

## 구성 | Structure

- 58개 PHP 페이지 (사용자 기능)
- 41개 관리자 페이지
- SQLite 데이터베이스
- Tailwind CSS 다크 테마
