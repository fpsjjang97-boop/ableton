## 1. 한 줄 소개

**MidiGPT는 작곡가의 DAW 안에서 함께 일하는, 음악 이론을 아는 AI 동료입니다.**

코드와 송폼을 이해하고, 변주·편곡·확장을 MIDI로 만들어 DAW로 즉시 보내줍니다.

---

## 2. 무엇을 하는가

### 2-1. 핵심 기능 5가지

**1) 변주 생성** — 16마디 곡 → 30초 안에 3가지 변주 후보
- 화성을 따른다 (off-scale 음 0%)
- 트랙별로 자연스러운 표현 (벨로시티 / CC / 아티큘레이션)

**2) 편곡 확장** — 멜로디만 → 베이스/드럼/패드/스트링 자동 추가
- 14개 트랙 카테고리 지원
- 원본 트랙 보존

**3) Audio → MIDI 변환** — wav 데모 → 편집 가능한 MIDI
- Demucs 6 stem 분리
- Basic Pitch 멜로디 추출

**4) Sheet → MIDI 변환** — PDF 악보 → MIDI
- SMT++ OMR 통합
- 변환 후 LLM 입력으로 즉시 사용 가능

**5) 스타일 LoRA 핫스왑** — 같은 입력, 다른 스타일
- 재즈 / 시티팝 / 메탈 / 클래식 등
- 5초 안에 LoRA 교체

### 2-2. 무엇이 아닌가

- ❌ 보컬 합성 (Suno/Udio 영역)
- ❌ 마스터링/믹싱 자동화
- ❌ 라이브 퍼포먼스 (별도 제품)
- ❌ 가사 생성
- ❌ 영상 편집

## 3. 누구를 위한 것인가

### 3-1. 1차 사용자 — 막힌 작곡가
| 항목 | 설명 |
|------|------|
| **직업** | 세미프로 작곡가, 음대 학생, 게임 BGM 프리랜서 |
| **도구** | Cubase / Ableton / Logic Pro |
| **페인** | "후렴구까지 만들었는데 브리지가 안 떠오름" |
| **우리가 주는 것** | DAW 안에서 한 번 클릭 → 변주 3개 → 그 자리에서 편집 |

### 3-2. 2차 사용자 — BGM 양산 작곡가
| 항목 | 설명 |
|------|------|
| **직업** | 게임/광고/유튜브 BGM 외주 |
| **페인** | 유사 분위기 5-10곡을 짧은 시간에 만들어야 함 |
| **우리가 주는 것** | 베이스 곡 + LoRA → 변주 5개 → 좋은 것만 다듬기 |

### 3-3. 3차 사용자 — 음악 이론 학습자
| 항목 | 설명 |
|------|------|
| **직업** | 음대생, 작곡 학원 수강생 |
| **페인** | 화성/송폼을 시각·청각적으로 동시에 이해 어려움 |
| **우리가 주는 것** | 코드 진행 입력 → 다양한 보이싱·변주 즉시 청취 |

## 4. 어떻게 쓰는가 — 첫 사용 5분 가이드

### Step 1: 설치 (1분)
```bash
git clone https://github.com/fpsjjang97-boop/ableton.git
cd ableton
pip install -r requirements.txt
python scripts/setup_check.py   # 8단계 PASS 확인
```

### Step 2: 첫 모델 다운로드 (1분)
```bash
# 학습된 base 체크포인트 자동 다운로드
# (정식 출시 후: 자동, 베타 단계: 수동 안내)
```

### Step 3: 첫 변주 생성 (3분)
```python
from midigpt.inference.engine import MidiGPTInference, InferenceConfig
from midigpt.tokenizer.encoder import SongMeta

inf = MidiGPTInference(InferenceConfig(
    model_path="./checkpoints/midigpt_ema.pt",
    lora_paths={"jazz": "./loras/jazz.pt"},
))

# 입력 MIDI 파일에서 변주 3개 생성
variations = inf.generate_variation(
    midi_path="my_song.mid",
    meta=SongMeta(key="C", style="jazz", section="chorus", tempo=120),
    num_return_sequences=3,
    repetition_penalty=1.1,
    no_repeat_ngram_size=4,
)

# 결과는 노트 딕셔너리 리스트
```

### Step 4: DAW로 보내기
- **Ableton Live**: `agents/ableton_bridge.py` 사용 (베타: 수동, 정식: 플러그인)
- **Cubase**: 생성 MIDI를 Drag & Drop
- **Logic Pro**: MIDI 파일 import

## 5. 화면별 설명 (JUCE 데스크톱 앱)

> 정식 출시 후 스크린샷 추가 예정. 현재 알파 빌드 기준.

### 5-1. 메인 윈도우 (5-zone Layout)
1. **상단 메뉴**: 파일 / LoRA / 설정
2. **좌측 파일 브라우저**: 입력 MIDI 선택
3. **중앙 피아노롤**: 입력 + 생성 결과 시각화
4. **우측 디테일 뷰**: 메타 정보 (key/tempo/style/section)
5. **하단 믹서/플레이 컨트롤**: 청취 + LoRA 핫스왑

### 5-2. 변주 생성 다이얼로그
- **Style**: jazz / city pop / metal / classical (LoRA 선택)
- **Variations**: 1~5 (num_return_sequences)
- **Temperature**: 0.7 ~ 1.3 (창의성 강도)
- **Min/Max length**: 토큰 단위 길이 통제
- **Generate** 버튼

### 5-3. 결과 비교 뷰
- 3개 변주를 나란히 청취
- 마음에 드는 것 선택 → DAW로 export
- 다른 것은 자동으로 "review/" 폴더에 저장 (학습 데이터로 활용)

## 6. 자주 묻는 질문 (FAQ)

### Q1. 어떤 DAW를 지원하나요?
- 현재: Ableton Live (Bridge 통합), Cubase 15 (어휘 통합)
- 알파 후: Cubase VST3 호스팅
- 1.0 후: Logic Pro, FL Studio

### Q2. 인터넷 연결이 필요한가요?
- **아니요.** 모든 추론은 로컬 GPU에서 실행됩니다.
- 클라우드 옵션은 Phase C(상용화)부터 선택 사항으로 제공.

### Q3. 어떤 GPU가 필요한가요?
- **권장**: RTX 3060 (12GB) 이상
- **최소**: RTX 2060 / GTX 1660 Ti (FP16, 8GB)
- CPU만으로도 동작하나 변주 1개 생성에 30초+ 걸림

### Q4. 모델은 어떻게 학습되었나요?
- **자체 50M decoder-only Transformer** (LLaMA 계보)
- **MIDI 전용 토크나이저** (448 tokens, 계층적 REMI)
- **Cubase 15 표현력** (32 articulation, CC, MPE-ready)
- **화성 마스킹** (생성 중 코드 스케일 외 음 자동 차단)
- 학습 데이터: 동업자 작곡가 자작곡 + 라이선스 free 외부 데이터셋

### Q5. 내 곡이 학습 데이터로 쓰이나요?
- **아니요.** 사용자 입력은 로컬에만 저장됩니다.
- 베타 사용자가 명시적으로 동의한 경우에만, 익명화된 형태로 학습에 활용 가능 (Phase C 이후 옵트인).

### Q6. 상업적으로 사용해도 되나요?
- 베타 단계: 비상업 사용만
- 정식 출시 후: Studio 라이선스 ($200/year, 5석) 구매 시 상업 사용 가능
- 생성된 MIDI의 저작권은 사용자에게 귀속

### Q7. 다른 AI 음악 도구와 뭐가 다른가요?
| 도구 | 출력 | DAW 통합 | 음악 이론 |
|------|------|----------|----------|
| MidiGPT | MIDI (편집 가능) | ✅ VST3 + Bridge | ✅ 화성 마스킹 |
| Suno / Udio | Audio (편집 불가) | ❌ | ❌ |
| Magenta | MIDI (데모 수준) | ❌ Ableton Pack 한정 | 부분 |
| ChatGPT 음악 | 텍스트 묘사만 | ❌ | ❌ (할루시) |

### Q8. 결과물이 어색해요. 왜 그런가요?
- **베타 단계에서는 학습 데이터가 부족합니다** (현재 ~54곡, 목표 2000+)
- 본격 학습이 끝나면 대부분의 어색함이 해소됩니다
- 특정 장르가 마음에 안 드시면 LoRA 어댑터 변경을 권장
- 피드백 채널: GitHub Issues 또는 Discord (베타 안내 참조)

## 8. 라이선스 / 가격

### 8-1. 라이선스
- **베타** (~Phase B 종료): 무료, 비상업 사용
- **정식 1.0** (Phase C): MIT 라이선스 base 모델 + 유료 LoRA / 구독

### 8-2. 가격 (예정)
| 티어 | 가격 | 포함 |
|------|------|------|
| Free | $0 | 데스크톱 앱, 베이스 모델, 무료 LoRA 1-2개, 비상업 |
| Pro | $15/월 | VST3 플러그인, 모든 LoRA, 무제한 변주, 상업 사용 |
| Studio | $200/year (5석) | Pro 모든 기능 + 5석 라이선스, 회사용 |
| API | 별도 견적 | B2B (게임 회사 BGM 자동화) |

### 8-3. LoRA 마켓플레이스 (Phase C)
- 외부 작곡가가 자기 스타일 LoRA를 학습/판매
- 수익 분배: 작곡가 70% / MidiGPT 30%
