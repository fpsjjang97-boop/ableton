## 구성 — 5 컷, 총 4분 30초

### 컷 1 — Hook (0:00 ~ 0:20, 20초)
**화면**: 빈 Cubase 트랙 + MidiGPT 플러그인 로드.

**나레이션 요지**:
> "DAW 안에서 AI 가 작곡 파트너가 됩니다. MidiGPT 는 VST3 플러그인이에요.
> 설치만 하면 Cubase, Ableton Live, 어디서든 동작합니다."

**시각 강조**: 플러그인 창이 호스트 트랙 인서트 슬롯에 떠오르는 장면 클로즈업.

---

### 컷 2 — MIDI 변주 생성 (0:20 ~ 1:40, 80초)
**화면**: 4 마디 피아노 리프 재생 → Generate 버튼 → MidiGPT 가 다음 16 마디 continuation 생성 → Cubase 타임라인에 노트 자동 기록 → 재생.

**나레이션 요지**:
> "기존에 치던 모티프에서 자연스럽게 이어 붙입니다.
> 스타일 드롭다운으로 city pop / jazz / metal 를 바꾸면 같은 모티프가 완전히 다른 느낌으로 재해석돼요.
> Temperature 슬라이더로 안전 / 모험 사이를 조절할 수 있습니다."

**포인트 강조**:
- Generate 후 **10초 내** 응답
- Undo / Export MIDI / Progress overlay 등 **UX 디테일** 1초씩 훑기
- 스타일 3개 교체를 빠르게 보여줘 LoRA 핫스왑 체감

---

### 컷 3 — 오디오 드롭 → MIDI (1:40 ~ 2:50, 70초)
**화면**: 기존 곡 `.wav` 를 플러그인 창으로 **드래그앤드롭** → "⚠ Beta" 라벨 + progress overlay → 30~60초 후 Input 피아노롤에 6 개 스템 (vocals/drums/bass/guitar/piano/strings) 색상 구분 표시.

**나레이션 요지**:
> "참고하고 싶은 레퍼런스 곡이 있으면 오디오 파일을 그냥 드롭하세요.
> 자동으로 드럼, 베이스, 피아노, 현악 트랙으로 분리되고 각각 MIDI 로 변환됩니다.
> Spotify Basic Pitch + ByteDance 피아노 모델이 내부적으로 돌아가요. Beta 단계라 편집이 필요할 수 있지만
> '아이디어 시드' 로는 충분히 유용합니다."

**포인트 강조**:
- Drag-enter hover 시 녹색 프레임 (UX 폴리쉬)
- 완료 후 dual-pane (Input / Output) 에서 변주 생성까지 연속 진행

---

### 컷 4 — 프리셋 + 크래시 복구 (2:50 ~ 3:50, 60초)
**화면**:
- "Save Preset" → "city_pop_chill" 저장
- Cubase 프로젝트 저장 → 닫음 → 재오픈 → 동일 상태 복원
- (데모용) 플러그인 호스트 강제 종료 → 재기동 시 "이전 세션 복구?" 다이얼로그 → 복구 클릭

**나레이션 요지**:
> "자주 쓰는 파라미터 조합은 프리셋으로 저장하고,
> 프로젝트 저장 시 MidiGPT 의 마지막 생성 결과까지 같이 남아요.
> 혹시 호스트가 크래시 나도 다음에 켜면 복구를 제안합니다."

**포인트 강조**: 실사용 안정감 — 작곡가가 날린 작업을 잃지 않음.

---

### 컷 5 — 차별화 + 콜투액션 (3:50 ~ 4:30, 40초)
**화면**: MidiGPT 로고 + 경쟁 제품 비교표 (간단). 웹사이트 / 가입 QR.

**나레이션 요지**:
> "Suno 나 Udio 는 오디오를 만들지만, MidiGPT 는 **편집 가능한 MIDI** 를 만듭니다.
> 작곡가가 여전히 주인인 도구예요.
> 6월 베타를 열고 있습니다. 링크에서 신청해주세요."
